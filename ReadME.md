A Python application with a modern Tkinter interface that allows users to **merge PDFs and images** efficiently. The app provides a user-friendly interface, drag-and-drop support, and customizable merging options.

## Features

* Merge multiple **PDF files** into a single PDF.
* Merge **images** into a PDF.
* **Drag and drop** support for easy file selection.
* **Custom output filename** and location.
* Modern and responsive **Tkinter UI**.
* Cross-platform: works on Windows, macOS, and Linux.

## Installation

1. Make sure you have **Python 3.10+** installed.
2. Clone this repository:

```bash
git clone https://github.com/AlexandreJustinRepia/AutoImagePdfMerger
cd AutoImagePdfMerger
```

3. Install required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python merge_pdfs.py
```

## Requirements

* Python 3.10+
* Tkinter (usually comes with Python)
* `PyPDF2` (for PDF merging)
* `Pillow` (for image handling)

You can install dependencies manually if needed:

```bash
pip install PyPDF2 Pillow
```

## Usage

1. Launch the app with `python merge_pdfs.py`.
2. Add PDFs or images to the list using the **Add Files** button.
3. Arrange the files in the desired order.
4. Click **Merge** to create a combined PDF.
5. Choose an output folder and filename.
6. Your merged PDF will be saved to the selected location.
